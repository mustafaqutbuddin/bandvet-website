# BANDVET Website

Modern, professional website for BANDVET pet care products.

## Features

- **Modern Design**: Glassmorphism, 3D effects, smooth animations
- **Brand Colors**: Blue (#2B7FD9), Yellow (#FDB840), Teal (#00C6B8)
- **Responsive**: Works on all devices
- **Interactive**: Scroll animations, hover effects, 3D product cards
- **Real Product Images**: Actual Bandvet Wrap and Dental Wipes photos

## Products Featured

1. **BANDVET WRAP** - Self-adhesive bandage wraps with paw prints
2. **BANDVET Dental Wipes** - 90-pack dog & cat dental finger wipes

## How to Use

1. Open `index.html` in your web browser
2. Scroll through the page to see all sections
3. Click "Buy on Amazon" buttons to visit product listings

## Sections

1. **Hero** - Eye-catching intro with tagline and product showcase
2. **Products** - Detailed cards for both products
3. **Features** - Why choose BANDVET
4. **Testimonials** - Customer reviews
5. **CTA** - Call to action
6. **Footer** - Links and info

## Customization

### To Update Product Images
Replace files in `/images/` folder with same names

### To Change Colors
Edit CSS variables in `styles.css`:
- `--primary-blue`
- `--secondary-yellow`
- `--accent-teal`

### To Add More Products
Copy a `.product-card-modern` section in `index.html` and update content
